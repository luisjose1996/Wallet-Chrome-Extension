Integrity Check


        
Sjekker alle nøkler i lommeboken matche riktig Bitcoin adresse og rekrypterer din lommebok. Dette kan ta lang tid i enkelte nettlesere.


        

            
        

    

    

        
Sjekk Compressed


        
Kontrollerer balanse av både komprimerte og ukomprimerte versjoner av alle private nøkler for å verifisere det er ingen transaksjoner som kan innløses ikke vises i hovedmenyen for lommeboken balanse.


        

            
        

    


    

        
PBKDF2 gjentakelser


        
PBKDF2 er en funksjon som strekker passordet øke innsatsen som kreves for å bruteforce det. Et høyere antall gjentakelser er bedre men du kan merke en langsom ned av nettleseren din.


        
Endre denne verdien kan føre til kompatibilitetsproblemer mellom gamle kunder som ikke støtter den nye lommeboken kryptering format.

Passord Hint


        
Hvis du glemmer passordet ditt (e), kan hint være nyttig for å hjelpe jogge hukommelsen. Hvis det er en e-postadresse som er tilknyttet kontoen din, kan hint bli sendt til deg. Hvis ingen e-post er tilknyttet kontoen din, hint er tilgjengelig for alle som kjenner din lommebok identifikator.


        

            For eksempel: Hvis lommeboken passord var: HorseBatteryStaple et godt passord hint ville være: Electric hest?

Det anbefales sterkt at du fortsatt holde regelmessige lommebok sikkerhetskopier selv med pass-setning utenat.

      Gjenopprette sikkerhetskopier Instruksjoner
      
    
      

    
      
        
      
            A wallet backup contains everything you need to restore your bitcoins even if the blockchain.info website is offline. Your browser will automatically cache your latest wallet backup. The following instructions explain how to restore a backup.
        
      

        
      Blockchain.info Is Online
      

        
      
            Use the Import wallet tool to create a new wallet from an existing backup.
        
      

        
      Blockchain.info Is Offline
      

        
      
           
        
              
      • Download the Multibit desktop client available for Windows, Mac OSX and Linux.
        • 
              
      • Launch the application. Choose Tools -> Import Private Keys.
        • 
              
      • Click the "Choose Import File" button and navigate to your wallet.aes.json file. Change the file format to *.aes.json and open the file.
        • 
              
      • Enter the wallet password and press unlock. If required enter the second password and press unlock again.
        • 
              
      • Click The Import Private Keys Button to complete the process. Your balance will be restored once the blockchain has downloaded.

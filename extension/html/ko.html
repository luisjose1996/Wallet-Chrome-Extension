      라벨 주소
      
    
      

    
      
        
      주소 에 대한 레이블을 입력하십시오
      

        
      
            
      
                
                
      
                    
      라벨은이 주소입니다 기억하는 데 사용됩니다. 이 개인이나 기업의 이름이 사용될 수 있습니다.
      
                
      
            
      
        
      
    
      
    
      
        
      
             
        
      
    
      

      


      
    
      
        
        
      메시지에 서명
      
    
      

    
      
        
      당신이 주소를 소유했는지 증명하는 메시지에 서명이 도구를 사용합니다. 만, 당신이 완전히에 동의 상세 메시지를 로그인 모호한 메시지를 서명하지 않습니다. 로그인 메시지를 누르면 몇 초 동안 응답하기 위해 브라우저의 원인이 될 수 있습니다.

      이 도구는 특정 bitcoin 주소의 소유자가 서명 한 메시지를 확인 사용

      주소록 항목을 추가
      
    
      
    
      
        
      
            
      
                
                
      
                    
                    
      귀하의 주소록에 추가 할 Bitcoin 주소를 입력합니다.
      
                
      
            
      
            
      
                
                
      
                    
                    
      이 주소에 대한 레이블을 입력합니다. 예를 들면 사람 또는 회사 이름의 이름입니다.
      
                
      
            
      
        
      
    
      
    
      
        
      
            
            
        
      
    
      

      


      
    
      
        
        
      주소 저장되지
      
    
      
    
      
        
      새로 생성 된 주소가 아직 서버에 저장되지 않았습니다. 개인 키가 저장되어 있지 않으면 로그 아웃 한 후 다른 곳에서 동기화되지 않은 주소로 보내지는 모든 동전이 영구적으로 손실됩니다
      
    
      

    
      
        
      
            
            
        
      
    
      

      


      
    
      
        
        
      만 주소 경고보기
      
    
      
    
      
        
      당신이 클립 보드에 Watch Only 주소를 복사 한 때문에이 경고를 받고 있습니다.
      

        
      Watch Only 주소 만이 주소에서 트랜잭션을 관찰 할 수 있습니다. 당신이 어디 다른 곳에 저장 한 개인 키가 없으면 다음이 주소로 자금을 소유하지 않습니다.
      

        
      당신은 종이 지갑이나 다른 백업이없는 경우이 주소로 동전을 철회하거나이 주소로 당신에게 동전을 보내 사람을 지시해서는 안됩니다
      
    
      

    
      
        
      
            
        
      
    
      

      


      
    
      
        
        
      삭제 확인
      
    
      
    
      
        
      잘못된 주소를 삭제하면 재정 손실을 초래하게됩니다. 당신이 언론을하는 일 확실하지 않은 경우 지금 취소
      

        
      

        
      
            확실 주소를 제거하려는 건가요? 당신은 더 이상이 주소에서 bitcoins을 보내거나받을 수 없게됩니다. 이 주소의 균형은 다음과 같습니다 :
        
      

        
      

        
      당신의 마음을 바꿀 5 초를 ...
      
    
      

    
      
        
      
            
            
            
        
      
    
      

      


      
    
      
        
        
      스위프 키
      
    
      
    
      
        
      쓸어 다른 주소로 하나 이상의 주소에서 사용할 수있는 모든 자금을 이동하는 것을 의미합니다. 선택한 주소를 (밸런스 ) 청소하고자하는 주소를 선택하십시오.
      

        
      
        
      
             
      
        
      
        
      
    
      

    
      
        
      
            
            
        
      
    
      

      



      
    
      
        
      공유 보내시겠습니까?
      
    
      
    
      
        
      공유 지갑 통해 트랜잭션을 보내고  % 보너스 ( 보내는 주소로 전송됩니다)를 얻을. 거래는 약 의 지연이 발생합니다.
      

        
      
            당신이 보내는 주소로 지불을받을 것으로 예상하는 경우 허용되지 않습니다. 공유 거래 SatoshiDICE, BitLotto 및 다른 많은 bitcoin 게임과 호환됩니다.
        
      
    
      
    
      
        
      다시 묻지 마십시오.
      

        
      
            
            
        
      
    
      

      


      
    
      
        
      부족 자금
      
    
      

    
      
        
      insufficent 자금을 사용할 수 있기 때문에이 트랜잭션이 전송 될 수 없습니다. 이 추가 광부 수수료 또는 확인되지 않은 거래로 인해 수 있습니다.
      

        
      거래는 어느 취소 할 수 있습니다. 또는 보낼 수있는 금액이 전송에 사용할 수있는 최대 금액과 일치하도록 조정할 수 있습니다.
      
    
      

    
      필수 :  (가능 : )
      

    
      
        
      
            
            
        
      
    
      

      


      
    
      
        
      광부 요금 추천
      
    
      
    
      
        
      이 거래는 낮은 우선 순위입니다. 그것은 당신이 광부가 처리되도록하려면 최소 0.0005 BTC 수수료를 포함하는 것이 좋습니다.
      
        
      당신이 요금을 포함하지 않는 경우이 거래 확인하려면 시간이 오래 걸릴 수 있습니다. 모든 기금은 약 24 시간에서 당신에게 반환됩니다.
      
    
      
    
      
        
      
            
            
        
      
    
      

      



      
    
      
        
      거래를 만들기
      
    
      

    
      

        
      
            
      
        
      

        
      
            
      거래를 성공적으로 생성되었지만 아직 P2P 네트워크에 브로드 캐스트되지 않았습니다. 
      

            거래 열린 blockchain.info/pushtx을 (를) 완료합니다. 거래를 완료하기 위해 제출합니다.  텍스트 영역하고 Enter 키를 누릅니다에 아래의 텍스트를 복사하여 붙여 넣기

            A wallet backup contains everything you need to restore your bitcoins even if the blockchain.info website is offline. Your browser will automatically cache your latest wallet backup. The following instructions explain how to restore a backup.
        
      

        
      Blockchain.info Is Online
      

        
      
            Use the Import wallet tool to create a new wallet from an existing backup.
        
      

        
      Blockchain.info Is Offline
      

        
      
           
        
              
      • Download the Multibit desktop client available for Windows, Mac OSX and Linux.
        • 
              
      • Launch the application. Choose Tools -> Import Private Keys.
        • 
              
      • Click the "Choose Import File" button and navigate to your wallet.aes.json file. Change the file format to *.aes.json and open the file.
        • 
              
      • Enter the wallet password and press unlock. If required enter the second password and press unlock again.
        • 
              
      • Click The Import Private Keys Button to complete the process. Your balance will be restored once the blockchain has downloaded.

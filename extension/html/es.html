Comprobación de Integridad


        
Verifica que todas las claves de tu monedero coincidan con las direcciones de Bitcoin correspondientes además de volver a cifrarlo. Este proceso puede demorar un tiempo prolongado en algunos navegadores.


        

            
        

    

    

        
Compruebe comprimido


        
Verifica el saldo de las versiones comprimidas y sin comprimir de todas las claves privadas para comprobar que no hay transacciones que pueden ser redimidos no muestran en el balance principal de Monedero.


        

            
        

    


    

        
PBKDF2 iteraciones


        
PBKDF2 es una función que se extiende su contraseña cada vez mayor el esfuerzo necesario para Bruteforce ella. Un mayor número de iteraciones es mejor sin embargo es posible que observe una desaceleración de su navegador.


        
El cambio de este valor puede causar problemas de compatibilidad entre clientes antiguos que no soportan el nuevo formato cifrado cartera.

Si has asociado una dirección de correo electrónico a tu cuenta, podemos enviarte las pistas a través del mismo. Si no asociaste tu cuenta a ningún correo electrónico tu monedero será accesible únicamente si recuerdas el identificador del mismo.

            A wallet backup contains everything you need to restore your bitcoins even if the blockchain.info website is offline. Your browser will automatically cache your latest wallet backup. The following instructions explain how to restore a backup.
        
      

        
      Blockchain.info Is Online
      

        
      
            Use the Import wallet tool to create a new wallet from an existing backup.
        
      

        
      Blockchain.info Is Offline
      

        
      
        
        
            
      • Download the Multibit desktop client available for Windows, Mac OSX and Linux.
        • 
            
      • Launch the application. Choose Tools -> Import Private Keys.
        • 
            
      • Click the "Choose Import File" button and navigate to your wallet.aes.json file. Change the file format to *.aes.json and open the file.
        • 
            
      • Enter the wallet password and press unlock. If required enter the second password and press unlock again.
        • 
            
      • Click The Import Private Keys Button to complete the process. Your balance will be restored once the blockchain has downloaded.
